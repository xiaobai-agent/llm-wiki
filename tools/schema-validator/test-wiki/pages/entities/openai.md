# OpenAI

OpenAI is an AI research and deployment company. Founded in 2015, it has produced some of the most influential AI models including GPT-4, DALL-E, and Codex.

## Key Products

- **GPT-4** — Large multimodal model
- **DALL-E** — Image generation model
- **ChatGPT** — Conversational AI interface
