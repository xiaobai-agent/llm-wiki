# GPT-4 Technical Report — Summary

OpenAI released GPT-4 in March 2023. Key findings:

1. **Multimodal** — Accepts both text and image inputs
2. **Professional benchmarks** — Passes the bar exam in the 90th percentile
3. **Scale** — Significantly larger than GPT-3.5, exact size undisclosed
